# Ejercicio práctico: operadores

## Instrucciones

Completa `exercise.py`.

El programa representa una compra y debe utilizar:

- operadores aritméticos;
- operadores de comparación;
- operadores lógicos;
- operadores de asignación.

## Datos

- Precio: `12.5`
- Cantidad: `8`
- Dinero disponible: `120`
- Stock inicial: `20`

## Resultados esperados

```text
Subtotal: 100.0
Vuelto: 20.0
Cajas completas: 2
Unidades sobrantes: 2
Potencia: 8
¿Tiene dinero suficiente?: True
¿Es una compra grande?: True
¿El pago es exacto?: False
¿La compra está aprobada?: True
¿Requiere revisión?: False
Stock final: 14
```
